# 🧑‍💻 Amatuers - 프로그래머스 데브코스 수강생들을 위한 커뮤니티

# 1. 프로젝트 소개
**Amatuers**는 프로그래머스 데브코스 수강생들을 위한 커뮤니티입니다. 다양한 코스의 사람들과 소통하고 정보를 공유합니다.

# 2. 핵심 기능
- **커뮤니티**
  - 자유게시판
  - 질문 / 토론
  - 블로그 공유 / 회고
- **함께해요(수강생 전용)**
  - 데브코스 장터
  - 팀원 구하기(스터디, 프로젝트 등)
  - 커피챗 / 멘토링
- **프로젝트 허브**
- **IT 정보**
  - IT 뉴스
  - 데브코스 후기
